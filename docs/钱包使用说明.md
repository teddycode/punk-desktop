## 版本说明
- V1.1：客户端已集成多种应用运行方式，包括
  - 页面级（vue页面，通过路由跳转和Router-link标签显示）
  - 网页级（任意框架的前端页面的链接，Iframe直接显示）
  - js程序级（任意js代码执行）
  - 可执行文件（任意可执行文件运行）
- V1.2：原生钱包和多钱包支持，包括
  - 集成内置浏览器miniBrowser，默认安装MetaMask钱包
  - 集成WalletConnect，支持钱包连接，全局调用

## 前置条件
1. 移动端安装MetaMask Mobile
   - 安卓端在[这里](https://github.com/MetaMask/metamask-mobile)下载安装
   - IOS的暂无，可使用WallectConnect官方在线的钱包[Demo](https://react-web3wallet.vercel.app/walletconnect)
2. 在移动端创建自己的私钥和账户
3. 领取测试网水龙头，以Goeril TestNet为例，请在[这里](https://goerlifaucet.com/)领取
4. 本次测试合约已部署好（USDT Constract address: 0x8942E52BE95b4f7e7566A91D77308F415DCf1d5D）
