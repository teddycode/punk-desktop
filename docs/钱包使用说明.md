# 钱包使用说明

## 当前支持

- 页面级应用：通过 WalletConnect 接入钱包
- 网页级应用：通过内置 mini browser 使用 MetaMask
- 两种方式目前不共享会话

## 前置条件

1. 安装可用钱包
   - Android：MetaMask Mobile
   - iOS：可使用 WalletConnect 官方 Demo 钱包联调，地址为 https://react-web3wallet.vercel.app/walletconnect
2. 创建测试账户并准备测试币
3. 确认测试合约地址

当前文档中的测试合约地址：

```text
0x8942E52BE95b4f7e7566A91D77308F415DCf1d5D
```

## 最小接入流程

### 1. 创建 WalletConnect 配置

```ts
import {
  createWeb3Modal,
  defaultConfig
} from '@punkos/ethers5/vue'

const modal = createWeb3Modal({
  projectId: 'your-project-id',
  chains: [
    {
      chainId: 5,
      name: 'Goerli',
      currency: 'ETH',
      explorerUrl: 'https://goerli.etherscan.io',
      rpcUrl: 'https://endpoints.omniatech.io/v1/eth/goerli/public'
    }
  ],
  ethersConfig: defaultConfig({
    metadata: {
      name: 'PunkOS',
      description: 'WalletConnect integration',
      url: 'https://www.punkos.com',
      icons: []
    },
    defaultChainId: 5,
    rpcUrl: 'https://cloudflare-eth.com'
  })
})
```

### 2. 获取钱包 Provider

```ts
import { useWeb3ModalProvider } from '@punkos/ethers5/vue'

const provider = useWeb3ModalProvider()
const walletProvider = provider.walletProvider.value
```

### 3. 创建 `ethers` 签名器

```ts
import { ethers } from 'ethers'

async function getSigner() {
  if (!walletProvider) {
    throw new Error('wallet disconnected')
  }

  const ethersProvider = new ethers.providers.Web3Provider(walletProvider)
  return ethersProvider.getSigner()
}
```

### 4. 读写合约

```ts
const USDT_ADDRESS = '0x8942E52BE95b4f7e7566A91D77308F415DCf1d5D'
const USDT_ABI = [
  'function balanceOf(address) view returns (uint)',
  'function transfer(address to, uint amount)'
]

async function getContract() {
  const signer = await getSigner()
  return new ethers.Contract(USDT_ADDRESS, USDT_ABI, signer)
}
```

- 读取数据：直接调用 `balanceOf`
- 写入数据：调用 `transfer` 等写方法后，移动端钱包会弹窗要求签名

## 常用组合函数

| 方法 | 用途 |
|------|------|
| `useWeb3Modal()` | 打开或关闭钱包连接弹窗 |
| `useWeb3ModalProvider()` | 获取当前钱包 Provider |
| `useWeb3ModalAccount()` | 获取当前连接账户 |
| `useWeb3ModalEvents()` | 监听连接、断开、切链等事件 |
| `useWeb3ModalTheme()` | 调整主题模式与变量 |

## 调试建议

- 钱包连接前先判断 `walletProvider` 是否存在
- 合约写操作需要用户在钱包内确认签名
- 调试事件流时，优先监听 `useWeb3ModalEvents()`
