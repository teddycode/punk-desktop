## 🎓 学习要点

### 对于开发者

1. **统一接口**: 所有环境使用相同的 API
   ```javascript
   const wallet = inject('wallet');
   ```

2. **单例模式**: 全局只有一个钱包实例
   ```javascript
   // 无论在哪里调用，都返回同一实例
   const wallet1 = inject('wallet');
   const wallet2 = useWalletService();
   // wallet1 === wallet2 ✅
   ```

3. **响应式状态**: 状态变化自动更新 UI
   ```javascript
   wallet.isConnected.value  // 响应式
   wallet.address.value      // 响应式
   ```

4. **跨环境支持**: 在任何地方都能使用
   - Vue 组件
   - 内嵌浏览器
   - iframe 页面
   - Dapp 详情页
