```javascript
setup() {

  return { wallet };
}
```

### 2.3 更新 initWallet 方法

替换原来直接调用 `createWeb3Modal` 的代码：

```javascript
async initWallet() {
  try {
    await this.wallet.initWallet(this.processUserInfo, this.userInfo);
    console.log('钱包初始化成功');
  } catch (e) {
    console.log('创建钱包错误：', e.toString());
  }
}
```

### 2.4 更新 login 方法

```javascript
async login() {
  const toast = useToast();
  try {
    const account = await this.wallet.openWallet();
    if (account) {
      toast.success(this.$t('toast.walletConnected'));
    }
  } catch (e) {
    toast.error('钱包连接失败: ' + e.message);
  }
}
```

### 2.5 更新 reLoginUser 方法

```javascript
async reLoginUser() {
  await this.deleteUserInfo();
  let res = await ipc.invoke('direct-logout', this.userInfo?.uid);
  if (res) {
    message.success(this.$t('message.logoutSuccess'));
    const success = await this.wallet.disconnectWallet();
    if (success) {
      this.login();
    }
  } else {
    message.error(this.$t('message.logoutFail'));
  }
}
```

## 📱 步骤 5: 在实际业务中集成（可选）

### 5.1 在 Dapp 详情页使用

创建或更新 Dapp 详情页组件：

```vue
<script setup>
import { inject, onMounted } from 'vue';

const wallet = inject('wallet');

onMounted(async () => {
  if (!wallet.checkInitialized()) {
    await wallet.initWallet();
  }
});
</script>

<template>
  <div class="dapp-details">
    <a-button v-if="!wallet.isConnected.value" @click="wallet.openWallet"> 连接钱包 </a-button>
    <div v-else>已连接: {{ wallet.address.value }}</div>
  </div>
</template>
```
