# 内嵌网页钱包测试 - 使用指南

> 如何通过图标打开钱包测试页面

## 📋 测试步骤

### 步骤 1: 创建测试图标

在桌面或图标管理中创建一个新图标，配置如下：

```javascript
{
  titleValue: '钱包测试',
  src: '/icons/wallet-test.png',  // 图标路径
  open: {
    type: 'internal',              // 使用内嵌浏览器
    value: 'wallet-test.html',     // 本地 HTML 文件
    wallet: true                   // 启用钱包功能
  }
}
```

### 步骤 2: 点击图标

点击创建的图标，系统会：

1. 调用 `browser.openInTable('wallet-test.html', { wallet: true })`
2. 初始化钱包服务
3. 如果未连接，弹出连接对话框
4. 打开浏览器页面
5. 加载 `wallet-test.html`

### 步骤 3: 测试钱包功能

在打开的页面中：

1. 查看环境检测结果
2. 点击"初始化钱包"
3. 点击"连接钱包"
4. 测试其他钱包操作

## 🐛 故障排除

### 问题 1: 页面打不开

**症状**: 点击图标后没有反应

**检查**:
1. 查看终端日志，是否有 `[Browser] openInTable 调用` 日志
2. 检查 `wallet-test.html` 是否在 `vite/html/` 目录
3. 检查 `vite/dist/` 目录是否有构建产物

**解决**:
```bash
cd vite
yarn run build
```

---

### 问题 2: 钱包服务未找到

**症状**: 页面显示"钱包服务不可用"

**检查**:
1. 查看浏览器控制台，是否有 `window.wallet` 对象
2. 检查是否注册了 `walletPlugin`
3. 查看 `[Browser] 钱包服务已获取` 日志

**解决**:
在 `main.ts` 中确保注册了插件：
```typescript
import walletPlugin from '@table/plugins/walletPlugin';
app.use(walletPlugin);
```

---

### 问题 3: URL 参数未传递

**症状**: 页面显示"URL wallet 参数: 无"

**检查**:
1. 查看 `[Browser] 添加 wallet 参数` 日志
2. 查看 `[Wallet Test] 当前 URL` 日志

**解决**:
确保 `browser.ts` 中正确添加了参数：
```typescript
if (isLocalHtml && options?.wallet) {
  const separator = url.includes('?') ? '&' : '?';
  finalUrl = `${url}${separator}wallet=true`;
}
```

---

### 问题 4: 本地 HTML 未转换

**症状**: 页面加载失败，显示 404

**检查**:
1. 查看 `[TableTabManager] 转换后的 URL` 日志
2. 检查 `render.getUrl()` 是否正常工作

**解决**:
确保 `tableTabManager.js` 中正确调用了 `render.getUrl()`:
```javascript
if (isLocalHtml) {
  const render = require('./render');
  finalUrl = render.getUrl(url);
}
```

## 💡 高级用法

### 自定义参数

可以在 URL 中添加更多参数：

```javascript
{
  open: {
    type: 'internal',
    value: 'wallet-test.html?wallet=true&autoConnect=true&network=mainnet',
    wallet: true
  }
}
```

在 HTML 中读取：
```javascript
const urlParams = new URLSearchParams(window.location.search);
const autoConnect = urlParams.get('autoConnect') === 'true';
const network = urlParams.get('network');
```

### 监听钱包初始化消息

在 HTML 中监听主进程发送的消息：

```javascript
if (typeof window.ipc !== 'undefined') {
  window.ipc.on('init-wallet', (event, data) => {
    console.log('收到钱包初始化消息:', data);
    if (data.enabled) {
      // 自动初始化钱包
      testInitWallet();
    }
  });
}
```
