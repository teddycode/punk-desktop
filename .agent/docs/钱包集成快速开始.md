## 🚀 快速开始

### 步骤 1: 注册钱包插件

在你的主入口文件（通常是 `main.ts` 或 `main.js`）中注册插件：

```typescript
import { createApp } from 'vue';
import App from './App.vue';
import walletPlugin from '@table/plugins/walletPlugin';

const app = createApp(App);

// 注册钱包插件
app.use(walletPlugin);

app.mount('#app');
```

### 步骤 2: 在 Splash.vue 中初始化

更新 `Splash.vue` 的 `initWallet` 方法：

```javascript
// 在 Splash.vue 中
import { inject } from 'vue';

export default {
  setup() {
    // 获取钱包服务
    const wallet = inject('wallet');
    return { wallet };
  },

  methods: {
    async initWallet() {
      try {
        // 使用统一的钱包服务
        await this.wallet.initWallet(this.processUserInfo, this.userInfo);
      } catch (e) {
        console.log('创建钱包错误：', e.toString());
      }
    },

    login() {
      const toast = useToast();
      // 使用统一的打开方法
      this.wallet.openWallet().then((account) => {
        if (account) {
          toast.success(this.$t('toast.walletConnected'));
        }
      });
    },
  },
};
```

### 步骤 3: 在任何组件中使用

#### 方式 1: 使用 inject (推荐)

```vue
<script setup>
import { inject } from 'vue';

const wallet = inject('wallet');

const handleConnect = async () => {
  const account = await wallet.openWallet();
  console.log('连接成功:', wallet.address.value);
};
</script>

<template>
  <div>
    <p v-if="wallet.isConnected.value">已连接: {{ wallet.address.value }}</p>
    <button v-else @click="handleConnect">连接钱包</button>
  </div>
</template>
```

#### 方式 2: 使用 this.$wallet (Options API)

```vue
<script>
export default {
  methods: {
    async connectWallet() {
      const account = await this.$wallet.openWallet();
      console.log('地址:', this.$wallet.address.value);
    },
  },
};
</script>
```

### 步骤 4: 在图标配置中启用钱包

在 icon 数据中添加 `wallet: true` 标识：

```javascript
{
  titleValue: 'Uniswap',
  src: '/icons/uniswap.png',
  open: {
    type: 'internal',
    value: 'https://app.uniswap.org',
    wallet: true  // 启用钱包功能
  }
}
```

## 📝 常见使用场景

### 场景 1: 在 DApp 详情页使用

```vue
<script setup>
import { inject, onMounted } from 'vue';
import { message } from 'ant-design-vue';

const wallet = inject('wallet');

onMounted(async () => {
  // 确保钱包已初始化
  if (!wallet.checkInitialized()) {
    await wallet.initWallet();
  }
});

const handleTransaction = async () => {
  if (!wallet.isConnected.value) {
    message.warning('请先连接钱包');
    await wallet.openWallet();
    return;
  }

  // 执行交易逻辑
  console.log('钱包地址:', wallet.address.value);
  console.log('链ID:', wallet.chainId.value);
};
</script>

<template>
  <div>
    <a-button v-if="!wallet.isConnected.value" @click="wallet.openWallet"> 连接钱包 </a-button>
    <div v-else>
      <p>已连接: {{ wallet.address.value }}</p>
      <a-button @click="handleTransaction">执行交易</a-button>
    </div>
  </div>
</template>
```

### 场景 2: 在内嵌浏览器中使用

```javascript
import browser from '@js/common/browser';

// 打开需要钱包的 DApp
browser.openInTable('https://app.uniswap.org', { wallet: true });
```

### 场景 3: 在 iframe 中使用

```html
<!-- iframe 页面 -->
<script>
  import { WalletClient } from '@js/common/walletBridge';

  const client = new WalletClient();

  // 连接钱包
  await client.connect();

  // 获取状态
  const status = await client.getStatus();
  console.log('钱包状态:', status);

  // 监听状态变化
  client.onStatusChange((status) => {
    console.log('状态更新:', status);
  });
</script>
```

## ⚠️ 注意事项

1. **单例模式**: 钱包服务全局只会初始化一次，避免重复创建
2. **初始化时机**: 建议在 Splash.vue 中完成初始化
3. **错误处理**: 务必添加 try-catch 处理钱包操作
4. **用户体验**: 连接失败时给予用户明确提示
5. **安全性**: 在 iframe 桥接时验证消息来源

## 🐛 常见问题

### Q1: 钱包未初始化

**A**: 确保在 Splash.vue 中调用了 `initWallet()`

### Q2: inject 返回 undefined

**A**: 确保已经注册了 walletPlugin 插件

### Q3: 钱包连接后刷新页面丢失状态

**A**: wallet store 已配置持久化，但钱包连接状态需要重新验证

### Q4: iframe 无法访问钱包

**A**: 确保父窗口已初始化 WalletBridge
