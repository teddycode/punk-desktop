## 🎯 快速使用

### 创建图标配置

```javascript
{
  titleValue: '钱包测试',
  src: '/icons/wallet.png',
  open: {
    type: 'internal',           // 内嵌浏览器
    value: 'wallet-test.html',  // 本地 HTML
    wallet: true                // 启用钱包
  }
}
```

### 代码调用

```javascript
import browser from '@js/common/browser';
browser.openInTable('wallet-test.html', { wallet: true });
```

### Q: 钱包服务不可用？
