## 🔄 数据流图

### 初始化流程

```
应用启动
    │
    ▼
main.ts 注册 walletPlugin
    │
    ▼
walletPlugin.install()
    │
    ├─► app.provide('wallet', walletService)
    └─► app.config.globalProperties.$wallet = walletService
    │
    ▼
Splash.vue mounted()
    │
    ▼
inject('wallet').initWallet()
    │
    ▼
createWeb3Modal(walletConfig())
    │
    ├─► 创建钱包实例（单例）
    └─► setupWalletListener()
    │
    ▼
钱包服务就绪 ✅
```

### 连接钱包流程

```
用户点击"连接钱包"
    │
    ▼
组件调用 wallet.openWallet()
    │
    ▼
检查钱包是否已初始化
    │
    ├─► 未初始化 ──► 调用 initWallet()
    └─► 已初始化 ──► 继续
    │
    ▼
打开 Web3Modal 对话框
    │
    ▼
用户选择钱包并授权
    │
    ▼
监听连接成功事件
    │
    ├─► 更新 walletService 状态
    ├─► 更新 Pinia store
    └─► 触发 toast 提示
    │
    ▼
关闭对话框
    │
    ▼
返回账户信息
    │
    ▼
钱包连接成功 ✅
```

### iframe 通信流程

```
父窗口                           iframe 页面
   │                                │
   ▼                                │
初始化 WalletBridge                 │
   │                                │
   ├─► 监听 message 事件             │
   │                                ▼
   │                          创建 WalletClient
   │                                │
   │                          ┌─────▼─────┐
   │                          │发送连接请求│
   │                          └─────┬─────┘
   │                                │
   │◄───────── postMessage ─────────┘
   │     { type: 'WALLET_CONNECT' }
   │
   ├─► 处理连接请求
   │
   ├─► 调用 walletService.openWallet()
   │
   ├─► 等待用户授权
   │
   ├─── postMessage ──────────────►│
   │   { type: 'WALLET_CONNECT_    │
   │     RESPONSE', data }          │
   │                                ▼
   │                          更新 iframe 状态
   │                                │
   │                                ▼
   │                          显示钱包信息 ✅
```
