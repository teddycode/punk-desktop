# 内嵌网页环境钱包集成流程分析

> 文档创建时间: 2026-01-30  
> 用途: 理清内嵌网页打开流程和钱包集成方案

## 📋 流程概述

内嵌网页的打开流程涉及 **3 个关键环节**：

```
icon.vue (点击图标)
    ↓
browser.openInTable() (浏览器工具)
    ↓
app/browser/index.vue (浏览器页面)
    ↓
ipc.send('addTableTab') (IPC 通信)
    ↓
tableTabManager.js (主进程)
    ↓
addTab() (创建 Electron 窗口)
    ↓
webContents.loadURL() (加载页面)
```

### 步骤 2: browser.openInTable 处理

**文件**: `vite/packages/table/js/common/browser.ts`

**功能**: 
1. 接收 URL 和选项参数
2. 如果启用钱包，初始化钱包服务
3. 打开浏览器页面

**代码逻辑**:
```typescript
async openInTable(url: string, options: { wallet?: boolean }) {
  // 1. 检查钱包参数
  if (options?.wallet) {
    // 2. 获取钱包服务
    const wallet = await this.getWalletService();
    
    // 3. 确保钱包已初始化
    if (!wallet.checkInitialized()) {
      await wallet.initWallet();
    }
    
    // 4. 如果未连接，自动连接
    if (!wallet.isConnected.value) {
      await wallet.openWallet();
    }
  }
  
  // 5. 打开浏览器页面
  this.openBrowserPage(url);
}
```

### 步骤 3: 打开浏览器页面

**文件**: `vite/packages/table/page/app/browser/index.vue`

**功能**: 
1. 显示浏览器 UI
2. 通过 IPC 通知主进程创建标签页

**代码位置**: `invokeAddTab` 方法

```javascript
async invokeAddTab(tab) {
  // 1. 准备参数
  let args = {
    position: this.getContentBounds(),  // 窗口位置
    url: tab.url,                       // 要加载的 URL
  };
  
  // 2. 发送 IPC 消息到主进程
  await ipc.send('addTableTab', JSON.parse(JSON.stringify(args)));
}
```

**关键点**:
- 这里只是发送消息，实际的窗口创建在主进程
- 参数包含窗口位置和 URL

### 步骤 4: 主进程接收 IPC 消息

**文件**: `src/main/tableTabManager.js`

**功能**: 
1. 监听 `addTableTab` 消息
2. 调用 `addTab` 创建 Electron 窗口
3. 返回结果给渲染进程

**代码逻辑**:
```javascript
ipc.on('addTableTab', async (event, args) => {
  console.log('需要添加tab', args);
  
  let tab = ;
  try {
    // 1. 创建标签页（Electron 窗口）
    let result = await this.addTab({ 
      url: args.url, 
      position: args.position 
    });
    
    tab = result.tab;
    console.log('完成添加', tab);
    
    // 2. 通知渲染进程创建成功
    this.sendToBrowser('addedTableTab', {
      tab: tab,
    });
  } catch (e) {
    console.error(e, '创建出错');
  }
});
```

### 步骤 5: 创建 Electron 窗口

**文件**: `src/main/tableTabManager.js`

**方法**: `addTab(args)`

**功能**: 
1. 创建 BrowserWindow
2. 加载指定 URL
3. 返回窗口信息

**代码逻辑**:
```javascript
async addTab(args) {
  // 1. 创建 BrowserWindow
  const win = new BrowserWindow({
    x: args.position.x,
    y: args.position.y,
    width: args.position.width,
    height: args.position.height,
    webPreferences: {
      nodeIntegration: false,
      contextIsolation: true,
      preload: path.join(__dirname, 'preload.js')
    }
  });
  
  // 2. 加载 URL
  if (args.url.startsWith('http')) {
    // 外部网页
    await win.loadURL(args.url);
  } else {
    // 内部页面（使用 render.getUrl）
    await win.loadURL(render.getUrl(args.url));
  }
  
  // 3. 返回窗口信息
  return {
    tab: {
      id: win.id,
      url: args.url,
      // ... 其他信息
    }
  };
}
```

## 🔑 关键点总结

### 1. URL 类型判断

```javascript
if (url.startsWith('http://') || url.startsWith('https://')) {
  // 外部网页：直接加载
  win.loadURL(url);
} else {
  // 内部页面：使用 render.getUrl
  win.loadURL(render.getUrl('app.html#/xxx'));
}
```

### 2. render.getUrl 的作用

**文件**: `main/render.js`

**功能**: 将相对路径转换为完整的本地文件 URL

```javascript
render.getUrl('app.html#/installApp')
// 转换为
// file:///path/to/vite/dist/app.html#/installApp
```

### 3. 钱包参数传递

**问题**: 如何将 `wallet: true` 参数传递到新窗口？

**方案**:
1. **URL 参数**: `app.html#/page?wallet=true`
2. **IPC 消息**: 窗口创建后发送初始化消息
3. **全局状态**: 使用 Pinia store 共享状态

## 🎯 钱包集成方案

### 方案 1: URL 参数传递（推荐）

**优点**: 简单直接，无需额外通信

**实现**:
```javascript
// icon.vue
browser.openInTable('wallet-test.html?wallet=true', { wallet: true });

// wallet-test.html
const urlParams = new URLSearchParams(window.location.search);
if (urlParams.get('wallet') === 'true') {
  // 初始化钱包
  initWallet();
}
```

### 方案 2: IPC 消息传递

**优点**: 更灵活，可以传递复杂数据

**实现**:
```javascript
// tableTabManager.js
async addTab(args) {
  const win = new BrowserWindow({...});
  await win.loadURL(render.getUrl(args.url));
  
  // 发送初始化消息
  if (args.wallet) {
    win.webContents.send('init-wallet', {
      enabled: true
    });
  }
}

// wallet-test.html (preload)
ipc.on('init-wallet', (event, data) => {
  if (data.enabled) {
    initWallet();
  }
});
```

### 方案 3: WalletBridge 桥接（最完整）

**优点**: 支持完整的钱包功能，状态同步

**实现**:
```javascript
// 父窗口（主应用）
const bridge = new WalletBridge();

// 子窗口（内嵌页面）
const client = new WalletClient();
await client.connect();
```
