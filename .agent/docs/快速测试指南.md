# 🚀 钱包集成测试 - 快速使用指南

> 5分钟快速上手钱包集成测试

## 📦 前置条件

确保以下内容已完成：

- ✅ 项目依赖已安装 (`yarn` 在根目录和 `vite/` 目录)
- ✅ 钱包插件已注册 (在 `main.ts` 中)
- ✅ 开发服务器已启动

## 🎯 快速开始

### 步骤 1: 启动开发服务器

```bash
# 在 vite 目录下
cd vite
yarn run dev
```

### 步骤 2: 访问测试中心

打开浏览器访问：
```
http://localhost:5173/test/wallet
```

（端口号可能不同，查看终端输出）

### 步骤 3: 按顺序测试

点击测试卡片，按照 **1 → 6** 的顺序进行测试：

```

  ↓
```

## 🐛 常见问题

### Q1: 测试页面打不开？

**A**: 检查路由是否已添加到 `route/first.ts`

### Q2: 钱包服务未注入？

**A**: 确认 `main.ts` 中已注册 `walletPlugin`

### Q3: MetaMask 未安装？

**A**: 
1. 安装 MetaMask 浏览器扩展
2. 或使用 WalletConnect 连接移动端钱包

### Q4: 连接失败？

**A**: 
1. 检查网络连接
2. 确认 MetaMask 已解锁
3. 查看浏览器控制台错误信息

### Q5: iframe 测试失败？

**A**: 
1. 确认 `test-wallet-client.html` 文件存在
2. 检查 WalletBridge 是否正确初始化
3. 查看控制台是否有跨域错误

## 💡 测试技巧

### 1. 观察日志

每个测试页面都有实时日志面板，仔细观察：
- ✅ 绿色 = 成功
- ❌ 红色 = 错误
- ⚠️ 黄色 = 警告
- ℹ️ 蓝色 = 信息

### 2. 使用浏览器开发工具

按 `F12` 打开开发者工具：
- **Console**: 查看详细日志
- **Network**: 查看网络请求
- **Application**: 查看 localStorage

- 拒绝钱包连接
